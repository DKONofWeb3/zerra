# Video content thumbnails

Drop video thumbnail images for the Top Performing Content table here.

## Naming

The component looks up images by `thumbId`. Try `.png` then `.jpg`
then `.webp` in order. Missing files render as a dark placeholder.

## Exporting from Figma

1. Select each thumbnail frame
2. Export as **PNG @ 2x** with 16:9 (or square; the component crops)
3. Drop into this folder with the filename above
