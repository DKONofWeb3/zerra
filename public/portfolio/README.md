# Portfolio page images

Drop bespoke artwork files here. The Portfolio page references them
by exact filename.

## Required files

| File              | Used for                                                |
| ----------------- | ------------------------------------------------------- |
| `paypal-hero.png` | The hand-holding-card hero image on the PayPal earnings card |
| `paypal-logo.png` | The PayPal app icon shown in the Payment Account card |

Supported extensions: `.png`, `.jpg`, `.webp`. The component tries
`.png` first; rename your export to match.

## Exporting from Figma

1. Select the hero image frame (the hand + card composition)
2. Export as **PNG @ 2x**
3. Drop into this folder as `paypal-hero.png`

## Missing files

If the file isn't found, the card shows a labeled placeholder so it's
obvious which file to drop in. The layout doesn't break.
